HTML

//Encabezado de primer nivel

HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de internet. Párrafo en html las etiquetas p>=apertura "Contenido" se utilizan para definir un párrafo img> = etiqueta para insertar una imagen src> = atributo que indica la ubicación de la imagen ejempor img src="imagen.jpg"> alt> = atributo que indica el texto alternativo de la imagen ejemplo img alt="texto alternativo" que indica el texto alternativo de la imagen que explica lo que se muestra en la imagen> alt> también se utiliza para mejorar la accesibilidad de la página web, para el uso de personas con discapacidad visual. Hay atributos comunes a todas las etiquetas, como el atributo id, que se utiliza para identificar de forma única un elemento en la página. Style = atributo que se utiliza para aplicar estilos CSS a un elemento.
